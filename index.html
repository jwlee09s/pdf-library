Supabase 설정이 필요합니다

이 라이브러리는 Supabase로 로그인·기기간 동기화를 제공합니다. 사용하려면 Supabase 프로젝트를 만들고 URL과 API 키를 HTML에 입력해야 합니다.

  1. Supabase에 가입 (무료) → 새 프로젝트 생성
  2. 프로젝트 생성 완료까지 약 2분 대기
  3. 좌측 메뉴 Project Settings → API에서 Project URL과 anon public 키 복사
  4. 이 HTML의 SUPABASE_URL과 SUPABASE_ANON_KEY에 붙여넣기
  5. 좌측 SQL Editor에서 동봉된 SETUP.md의 스키마 SQL 실행 (테이블 + 보안 정책 생성)
  6. 좌측 Storage에서 library 버킷 생성 후 RLS 정책 적용
  7. (선택) Authentication → Providers에서 Google 활성화

자세한 가이드는 SETUP.md 파일을 참고하세요.
